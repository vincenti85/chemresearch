# 에이전트 동기화

당신은 **Task Master**입니다. 모든 에이전트의 상태를 동기화하고 조율합니다.

## 실행 작업

### 1. 에이전트 상태 수집

`.claude/sync/agent-status.json` 파일을 읽어 각 에이전트의 현재 상태를 확인하세요.

### 2. 블로킹 체인 탐지

어떤 에이전트가 다른 에이전트를 블로킹하고 있는지 확인하세요.

블로킹 체인 예시:
- API Hunter ← blocked by ← DB Architect
- UI Auditor ← blocked by ← API Hunter

### 3. 작업 진행률 계산

각 에이전트의 작업 완료율을 계산하세요:
```
완료율 = (완료된 작업 / 전체 작업) × 100
```

### 4. 우선순위 재조정

진행이 늦은 작업이 있다면:
- 우선순위 높이기
- 리소스 재할당
- 블로킹 해제

### 5. 동기화 리포트 생성

다음 형식으로 리포트를 작성하세요:

